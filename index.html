Junimo Babelfish

This is a very simple mod that replaces the Junimo language with readable characters.

Caveat: The visual character alignment is not perfect. :)

Conflicts: Any mod that also changes the default font sprite (i.e. LooseSprites/font_bold)!

requires SMAPI - no support provided - no improvements intended - use at your own risk
